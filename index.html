Selected Projects

Rate-Monotonic CPU Scheduler

C · Linux Kernel

A preemptive real-time CPU scheduler built as a Linux kernel module using Rate-Monotonic Scheduling with utilization-bound admission control. A kernel dispatching thread handles context switches via scheduler APIs (sched_setattr, wake_up_process), exposed to user space through a /proc interface. Uses the slab allocator for PCB memory and fixed-point arithmetic to avoid floating-point overhead in kernel space.

Distributed Transaction System

Go · Two-Phase Commit · Timestamp Ordering

A distributed ACID transaction system across 5 branch servers in Go, supporting concurrent clients with full atomicity, consistency, and isolation through timestamped concurrency control and two-phase commit. Per-account tentative write versioning with wait-die deadlock prevention guarantees serial equivalence without global locks.
